Writing a Dockerfile

A Dockerfile is a list of instructions. Each instruction creates a layer in the image.

# syntax=docker/dockerfile:1.7
FROM node:20-alpine           # base image
WORKDIR /app                  # cd into /app for subsequent commands
COPY package*.json ./         # copy dependency manifests first (better caching)
RUN npm ci --omit=dev         # install deps
COPY . .                      # copy source code
ENV NODE_ENV=production
EXPOSE 3000
CMD ["node", "server.js"]     # default command
docker build -t myapp:1.0 .          # build from current dir
docker run -p 3000:3000 myapp:1.0    # run it

Volumes & Data

Containers are ephemeral — data inside is lost when removed. Use volumes for persistence.

# Named volume (managed by Docker)
docker volume create db-data
docker run -d -v db-data:/var/lib/postgresql/data postgres:16

# Bind mount (a host directory mounted in)
docker run -d -v $(pwd)/src:/app/src myapp:1.0

docker volume ls
docker volume inspect db-data
Prefer named volumes over bind mounts for databases — they're easier to back up, migrate, and don't depend on host paths.

Networking Basics

docker network ls                       # list networks
docker network create mynet             # create a custom bridge
docker run -d --network mynet --name api myapi:1.0
docker run -d --network mynet -p 8080:80 --name web nginx   # web→api via DNS name "api"
docker network inspect mynet
DNS magic: on a user-defined network, containers resolve each other by name (here, api is reachable from web as http://api:80).

Multi-stage Builds

Use multiple FROM stages to keep build tools out of the final image.

# syntax=docker/dockerfile:1.7
# ---- build stage ----
FROM golang:1.22 AS build
WORKDIR /src
COPY go.mod go.sum ./
RUN go mod download
COPY . .
RUN CGO_ENABLED=0 go build -o /out/app

# ---- runtime stage ----
FROM gcr.io/distroless/static-debian12:nonroot
COPY --from=build /out/app /app
USER nonroot:nonroot
ENTRYPOINT ["/app"]
A Go binary from this Dockerfile is ~10–20 MB instead of ~800 MB — and has no shell, no package manager, and a non-root user by default.

Don't bake secrets into images. Anything in a layer is in the image forever — even after you "delete" it in a later layer. Use env vars, secrets, or mount at runtime.

Security Hardening

  1. Run as non-root. Add USER in your Dockerfile.
  2. Minimal base images. Prefer alpine, distroless, or scratch.
  3. Don't bake secrets. Use BuildKit secrets, env vars at runtime, or a secret manager.
  4. Pin versions. Avoid latest in production — pin by digest for reproducibility.
  5. Read-only filesystems. docker run --read-only --tmpfs /tmp myapp
  6. Drop capabilities. --cap-drop=ALL --cap-add=NET_BIND_SERVICE
  7. Scan images. docker scout cves myapp:1.0 or trivy image myapp:1.0.
docker run --rm \
  --read-only \
  --tmpfs /tmp \
  --cap-drop=ALL \
  --security-opt no-new-privileges \
  --user 1000:1000 \
  myapp:1.0

⚡ Asynchrony

Why this matters: Enterprise integrations involve multiple API calls to different systems. Knowing how to handle these efficiently is critical for performance and reliability.

Retry with exponential backoff

// Modern retry with exponential backoff using recursion
const fetchWithRetry = async (url, options = {}, maxRetries = 3) => {
  const sleep = (ms) => new Promise(resolve => setTimeout(resolve, ms));

  const attempt = async (retriesLeft) => {
    try {
      const response = await fetch(url, options);
      if (!response.ok) throw new Error(response.status);
      return response.json();
    } catch (error) {
      if (retriesLeft === 0) throw error;
      const delay = 1000 * (2 ** (maxRetries - retriesLeft));
      console.log(`Retry ${maxRetries - retriesLeft + 1}/${maxRetries} after ${delay}ms`);
      await sleep(delay);
      return attempt(retriesLeft - 1);
    }
  };

  return attempt(maxRetries);
};

// Shorter version using a loop
const fetchWithRetryShort = async (url, options = {}, maxRetries = 3) => {
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      const response = await fetch(url, options);
      if (!response.ok) throw new Error(response.status);
      return response.json();
    } catch (error) {
      if (attempt === maxRetries) throw error;
      await new Promise(r => setTimeout(r, 1000 * 2 ** attempt));
    }
  }
};

class IntegrationError extends Error {

  constructor(message, code, system) {
    super(message);

  }

  name = 'ValidationError';
  fieldErrors = [];

    super(message);

Structured error handling

// Modern error handling with arrow function + early returns
const syncAccount = async (accountData) => {
  try {
    const result = await createAccountInCRM(accountData);
    return { success: true, crmId: result.id };
  } catch (error) {
    // Pattern match using instanceof
    if (error instanceof ValidationError) {
      console.error(`Validation failed: ${error.fieldErrors}`);
      return { success: false, type: 'validation', errors: error.fieldErrors };
    }

    // Handle specific error codes
    if (error.code === 'NETSUITE_DUPLICATE') {
      console.error(`Account already exists: ${accountData.externalId}`);
      return { success: false, type: 'duplicate' };
    }

    // Retry with backoff
    if (error instanceof RateLimitError) {
      console.log(`Waiting ${error.retryAfter}s...`);
      await new Promise(r => setTimeout(r, error.retryAfter * 1000));
      return syncAccount(accountData);
    }

    // Re-throw unknown errors
    throw error;
  }
};

// Modern approach: Result pattern (like Rust)
const safeSync = async (accountData) => {
  try {
    return { ok: true, data: await createAccountInCRM(accountData) };
  } catch (error) {
    return { ok: false, error };
  }
};

Circuit breaker pattern

// Modern Circuit Breaker with private fields (#)
class CircuitBreaker {
  #state = 'CLOSED';
  #failureCount = 0;
  #failureThreshold;
  #resetTimeout;

  constructor(failureThreshold = 5, resetTimeout = 30000) {
    this.#failureThreshold = failureThreshold;
    this.#resetTimeout = resetTimeout;
  }

  get state() { return this.#state; }

  async execute(fn) {
    if (this.#state === 'OPEN') {
      throw new Error('Circuit OPEN - too many failures');
    }

    try {
      const result = await fn();
      this.#onSuccess();
      return result;
    } catch (e) {
      this.#onFailure();
      throw e;
    }
  }

  #onSuccess() {
    this.#failureCount = 0;
    this.#state = 'CLOSED';
  }

  #onFailure() {
    this.#failureCount++;
    if (this.#failureCount >= this.#failureThreshold) {
      this.#state = 'OPEN';
      setTimeout(() => this.#state = 'HALF_OPEN', this.#resetTimeout);
    }
  }
}

// Usage
const nsBreaker = new CircuitBreaker();
const result = await nsBreaker.execute(() => fetch('/api/ns'));

// Modern functional approach (factory function)
const createBreaker = (threshold = 5, timeout = 30000) => {
  let failures = 0;
  let state = 'CLOSED';

  return {
    get state() { return state; },
    execute: async (fn) => {
      if (state === 'OPEN') throw new Error('OPEN');
      try { const r = await fn(); failures = 0; state = 'CLOSED'; return r; }
      catch { failures++; if (failures >= threshold) state = 'OPEN'; throw; }
    }
  };
};
Logging tip: Always log the correlation ID, system name, and timestamp. Makes debugging integrations much easier.

🔄 Data Mapping

Field mapping function

// Map NetSuite fields to CRM fields (modern arrow functions)
const fieldMapping = {
  // NetSuite field → CRM field
  'companyname': 'name',
  'email': 'primaryEmail',
  'phone': 'primaryPhone',
  'custentity_billing_cycle': 'billingCycle',
  'custentity_external_id': 'erpId'
};

// Modern functional approach using reduce
const mapNetSuiteToCRM = (nsRecord) => {
  return Object.entries(fieldMapping).reduce((acc, [nsField, crmField]) => {
    if (nsRecord[nsField] !== undefined) {
      acc[crmField] = nsRecord[nsField];
    }
    return acc;
  }, {});
};

// Even shorter: one-liner with filter
const mapNetSuiteToCRMShort = (nsRecord) =>
  Object.fromEntries(
    Object.entries(fieldMapping)
      .filter(([nsField]) => nsRecord[nsField] !== undefined)
      .map(([nsField, crmField]) => [crmField, nsRecord[nsField]])
  );

Data transformation

// Modern transformers using method shorthand + arrow functions
const transformers = {
  // Date: NetSuite '12/15/2024' → ISO '2024-12-15T00:00:00Z'
  date: (value) => {
    if (!value) return null;
    const [month, day, year] = value.split('/');
    return new Date(year, month - 1, day).toISOString();
  },

  // Currency: '1500.00' → 1500 (number)
  currency: (value) => (typeof value === 'string' ? parseFloat(value) : value),

  // Boolean: 'T'/'F' → true/false (modern ternary chain)
  boolean: (value) =>
    value === 'T' || value === 'true' ? true :
    value === 'F' || value === 'false' ? false : null,

  // Custom list: '1-CUSTOMER' → 'CUSTOMER'
  list: (value) => value?.split('-')[1] ?? null
};

// Modern transform with default parameter + optional chaining
const transformValue = (value, type, defaultVal = value) =>
  transformers[type]?.(value) ?? defaultVal;

// Usage with nullish coalescing
const result = transformValue('1500.00', 'currency'); // → 1500

Idempotency keys — prevent duplicate syncs

// Modern arrow function for generating idempotency keys
const generateIdempotencyKey = (source, recordType, externalId) =>
  `${source}:${recordType}:${externalId}`;

// Check if we've already synced this record
const syncAccount = async (nsAccount) => {
  const key = generateIdempotencyKey('netsuite', 'account', nsAccount.internalId);
  const cacheKey = `sync:${key}`;

  // Check if already synced (early return pattern)
  const existing = await redis.get(cacheKey);
  if (existing) {
    console.log(`Already synced ${key}, skipping`);
    return { skipped: true, existingId: existing };
  }

  // Proceed with sync
  const crmResult = await createCRMAccount(mapNetSuiteToCRM(nsAccount));

  // Store with 30-day TTL
  await redis.set(cacheKey, crmResult.id, 'EX', 86400 * 30);

  return { synced: true, crmId: crmResult.id };
};

// Modern: Use SET with NX for atomic idempotency
const syncAccountAtomic = async (nsAccount) => {
  const key = `sync:${generateIdempotencyKey('netsuite', 'account', nsAccount.internalId)}`;
  const result = await redis.set(key, '', 'NX', 'EX', 86400 * 30);

  if (!result) {
    return { skipped: true, message: 'Already being synced' };
  }

  // Do sync work...
  return { synced: true };
};

Parent-child record sync

// Sync Account first, then Contacts (modern arrow function)
const syncAccountWithContacts = async (nsAccount) => {
  // 1. Sync parent record first
  const crmAccount = await syncAccount(nsAccount);
  if (!crmAccount.success) throw new Error('Account sync failed');

  // 2. Get contacts from NetSuite
  const nsContacts = await fetchContacts(nsAccount.internalId);

  // 3. Sync each contact with Promise.allSettled
  const results = await Promise.allSettled(
    nsContacts.map((contact) => syncContact({ ...contact, accountId: crmAccount.crmId }))
  );

  // 4. Analyze results (modern filter + concise callback)
  const failed = results.filter(({ status }) => status === 'rejected');
  if (failed.length > 0) {
    console.error(`${failed.length} contacts failed to sync`);
  }

  return {
    account: crmAccount,
    contactsSynced: results.length - failed.length
  };
};

// Modern: Process children sequentially for better error handling
const syncWithSequentialChildren = async (nsAccount) => {
  const crmAccount = await syncAccount(nsAccount);
  const nsContacts = await fetchContacts(nsAccount.internalId);

  let synced = 0;
  for (const contact of nsContacts) {
    try {
      await syncContact({ ...contact, accountId: crmAccount.crmId });
      synced++;
    } catch (e) {
      console.error(`Failed to sync contact ${contact.id}: ${e.message}`);
    }
  }

  return { account: crmAccount, contactsSynced: synced };
};
Always sync parent records before children. If the parent fails, skip the children — it prevents orphaned records in the target system.
NetSuite Account → Sync Parent First → Get Contacts → Link to Parent
Idempotency: Use a combination of source system + record type + external ID as your key. This ensures the same record doesn't get synced twice even if the sync runs multiple times.

OAuth 2.0 token refresh

// Modern token cache with arrow function + closure
const createTokenCache = () => {
  let cachedToken = null;

  const getValidToken = async () => {
    // Early return if token is still valid (with buffer)
    if (cachedToken?.expiresAt > Date.now()) {
      return cachedToken.accessToken;
    }

    // Refresh token
    const response = await fetch('/oauth/token', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        grant_type: 'refresh_token',
        refresh_token: cachedToken?.refreshToken
      })
    });

    // Update cache with buffer
    cachedToken = {
      ...(await response.json()),
      expiresAt: Date.now() + 3500 * 1000
    };

    return cachedToken.accessToken;
  };

  const clearToken = () => { cachedToken = null; };

  return { getValidToken, clearToken };
};

// Usage
const tokenCache = createTokenCache();
const token = await tokenCache.getValidToken();

Handle pagination

// Modern pagination with async generator (ES2022)
async function* fetchAllRecords(endpoint, pageSize = 100) {
  let offset = 0;

  while (true) {
    const response = await fetch(
      `${endpoint}?limit=${pageSize}&offset=${offset}`
    );
    const { records, hasMore } = await response.json();

    yield* records;

    if (!hasMore) break;
    offset += pageSize;
  }
}

// Usage with for await...of
for await (record of fetchAllRecords('/api/accounts')) {
  console.log(record);
}

// Classic approach (still useful)
const fetchAllRecordsClassic = async (endpoint, pageSize = 100) => {
  const allRecords = [];
  let offset = 0;
  let hasMore = true;

  while (hasMore) {
    const { records, hasMore: more } = await fetch(
      `${endpoint}?limit=${pageSize}&offset=${offset}`
    ).then(r => r.json());

    allRecords.push(...records);
    hasMore = more;
    offset += pageSize;
  }

  return allRecords;
};

Verify webhook signatures

// Modern webhook verification with arrow functions
import { createHmac, timingSafeEqual } from 'crypto';

const verifyWebhookSignature = (payload, signature, secret) => {
  const expected = createHmac('sha256', secret)
    .update(payload)
    .digest('hex');

  return timingSafeEqual(
    Buffer.from(signature),
    Buffer.from(expected)
  );
};

// Modern Express handler with async/await
app.post('/webhook/crm', async (req, res) => {
  const { ['x-webhook-signature']: signature } = req.headers;

  const payload = JSON.stringify(req.body);

  if (!verifyWebhookSignature(payload, signature, process.env.WEBHOOK_SECRET)) {
    return res.status(401).send('Invalid signature');
  }

  // Process webhook asynchronously
  handleWebhookEvent(req.body);
  res.status(200).send('OK');
});

// Modern: Return early for invalid signature

Batch operations

// Modern batch processing with arrow functions
const batchSync = async (records, batchSize = 50, delayMs = 1000) => {
  const sleep = (ms) => new Promise(resolve => setTimeout(resolve, ms));
  const results = [];

  for (let i = 0; i < records.length; i += batchSize) {
    const batch = records.slice(i, i + batchSize);
    const batchNum = Math.floor(i / batchSize) + 1;
    console.log(`Processing batch ${batchNum}...`);

    const batchResults = await Promise.allSettled(
      batch.map(record => syncRecord(record))
    );
    results.push(...batchResults);

    // Rate limit: wait between batches (skip on last batch)
    if (i + batchSize < records.length) {
      await sleep(delayMs);
    }
  }

  return results;
};

// Modern: Chunk-based processing with error aggregation
const batchSyncModern = async (records, batchSize = 50) => {
  const chunks = Array.from({ length: Math.ceil(records.length / batchSize) },
    (_, i) => records.slice(i * batchSize, (i + 1) * batchSize)
  );

  const results = await Promise.allSettled(
    chunks.map(chunk => Promise.allSettled(chunk.map(syncRecord)))
  );

  // Flatten and filter failures
  const failures = results.flatMap(r => r.status === 'rejected' ? [r.reason] : []);

  return { total: records.length, failures };
};

🏢 NetSuite Integration

REST API authentication

// NetSuite REST API with token-based auth (TBA)
const netsuiteRequest = async (endpoint, method = 'GET', body = null) => {
  const { NETSUITE_ACCOUNT_ID: accountId, NETSUITE_TOKEN_ID: tokenId, NETSUITE_TOKEN_SECRET: tokenSecret } = process.env;

  // Generate TBA signature
  const nonce = Math.random().toString('base64').slice(2, 22);
  const timestamp = Math.floor(Date.now() / 1000);
  const signature = btoa(`${tokenId}&${timestamp}&${nonce}&${method}&${endpoint}&${accountId}`);

  const baseUrl = `https://${accountId}.suitetalk.api.netsuite.com/services/rest/record/v1`;
  const headers = {
    'Authorization': `TBA-V1 Ha=${accountId}, Haid=${tokenId}, Hats=${timestamp}, Haon=${nonce}, Hs=${signature}`,
    'Content-Type': 'application/json',
    'Accept': 'application/json'
  };

  const response = await fetch(`${baseUrl}/${endpoint}`, { method, headers, body: body ? JSON.stringify(body) : undefined });

  if (!response.ok) {
    const error = await response.json();
    throw new IntegrationError(error.message || 'NetSuite API error', `NS_${response.status}`, 'netsuite');
  }

  return response.json();
};

Internal ID mapping

// Modern IdMapper with private fields and arrow functions
class IdMapper {
  #cache;
  #ttl = 86400 * 30; // 30 days

  constructor(cache) {
    this.#cache = cache;
  }

  #key = (system, externalId) => `idmap:${system}:${externalId}`;

  async getInternalId(system, externalId) {
    return await this.#cache.get(this.#key(system, externalId));
  }

  async setMapping(system, externalId, internalId) {
    await this.#cache.set(this.#key(system, externalId), internalId, 'EX', this.#ttl);
  }

  async getOrFetch(system, externalId, fetcher) {
    // Try cache first, fetch if not found
    const cached = await this.getInternalId(system, externalId);
    if (cached) return cached;

    // Fetch from source and cache
    const { internalId } = await fetcher(externalId);
    await this.setMapping(system, externalId, internalId);
    return internalId;
  }
}
